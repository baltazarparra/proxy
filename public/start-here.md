## What guia is

guia is a public methodology for structured software development with code agents.

It has two main uses:

1. Read it as a guide and consult specific sections, templates, and references.
2. Hand it to a code agent so the agent can bootstrap and follow a more structured workflow.

## Recommended entrypoints
